# VTM Midia

Portfolio institucional da VTM Midia, agencia de marketing e produtora audiovisual de Teresopolis, RJ.

## Como visualizar

Abra `index.html` no navegador.

## Assets principais

- `assets/vtm-logo.png`: logo original enviada.
- `assets/vtm-logo-crop.png`: versao recortada para cabecalho.
- `assets/clientes-vtm.png`: painel de clientes.
- `assets/hero-agencia.png`: imagem de apoio do hero.
